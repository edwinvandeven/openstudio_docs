Prerequisites
=============

**General**

* Python 2.7 (Not Python 3)
* MySQL 5.5 or later database
* Redis server
* Web2py (http://www.web2py.com)

**Python Modules**

* openpyxl
* html2text
* pytz
* redis
* mollie-api-python < 2.0 (e.g. pip install -Iv mollie-api-python==1.4.2)
* weasyprint
* Pillow
* pybarcode
* qrcode
* mailchimp3
